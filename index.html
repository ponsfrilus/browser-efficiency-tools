Partie 2 — User Scripts

A userscript (or user script) is a program, usually written in JavaScript, for modifying web pages to augment browsing.
Permet de modifier n'importe quelle page...
De la manière qu'on a envie...
Et d'en relever le goût !

Comment ça marche ?

Installer un extension du navigateur permettant de gérer les scripts
  • GreaseMonkey (Firefox)
  • TaperMonkey (Chrome)
  • ViolentMonkey (Opera)
  • etc...
puis installer des scripts (OpenUserJS)
ou créer les siens !
